# 🦈 Chompy Meme Token Landing

💼 **Commercial Project**

🔗 Live Demo: https://pashko911.github.io/chompy/

---

## ✨ About

**Chompy** is a promotional landing page for a meme token project built around a playful underwater visual concept.  
The website is designed to introduce the token, explain how to buy it, present tokenomics, and drive users toward key conversion actions.

The project combines strong themed art direction with animated UI elements and a scroll-based storytelling structure tailored for crypto audiences.

---

## ⚡ Highlights

- Responsive landing page for desktop and mobile devices
- Large layered hero section with custom illustrated assets
- Thematic underwater art direction across all sections
- Step-by-step "How to Buy" block for onboarding users
- Animated scroll interactions powered by watchers
- Infinite partners/PFP marquee section
- Tokenomics section with highlighted supply and tax details
- Conversion-focused CTA blocks and social links integration
